Datenschutzerklärung – Translation Bot

Der Translation Bot für Discord dient ausschließlich dem Zweck, Nachrichten automatisch zu übersetzen.

Welche Daten werden verarbeitet?

  • Nachrichteninhalte aus bestimmten Discord-Kanälen
  • Benutzername (nur zur Anzeige, keine Speicherung)

Was passiert mit den Daten?

Die übermittelten Nachrichten werden über eine sichere Verbindung zur Übersetzung an die DeepL API weitergeleitet und danach gelöscht. Es werden keine Daten gespeichert.

Wer hat Zugriff auf die Daten?

Nur autorisierte Botsysteme erhalten Zugriff auf Nachrichten in freigegebenen Channels. Es erfolgt keine Weitergabe an Dritte.
